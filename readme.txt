Небольшой путеводитель по множеству файлов проекта
---------------------------------------------------------------------------------
Скрипты
 /blocks.js/-- хранит в себе классы различных блоков.
 /utils.js/-- хранит в себе шаблонные обёртки, а также задаёт стиль через функцию css()
 /index.js/-- своеобразная "точка сбора" для всего html содержания сайта и отправки в /index.html/
 /model.js/-- содержить в себе все элементы которые потребуется отобразить на страничке, то есть здесь происходит создание
экземпляров классов
 /site.js/-- собирает всё, что будет отображено на сайте и при необходимости выдаёт по запросу
 /parcer.js/-- извлекает данные из localStorage данные по ключу
 /tables-utils.js/-- содержить в себе шаблоны для разных таблиц(название таблицы, название столбцов, стили и прочий html)
 /tables-builder.js/-- производит сборку таблицы, получая ссылку на данные таблицы
 /editor.js/-- редактирует данные, получая ключ
------------------------------------------------------------------------------------
CSS-файлы
/main.css/-- содержит в себе все стили
